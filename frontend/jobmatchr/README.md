## Setup Instructions

**Make sure you have yarn globally installed.** Alternatively you can use npm.

*Install dependencies:*
``` 
yarn 
```

or

```
npm install
```

### Running for Production/Demo:

```
yarn production
```

### Running for Local Development: 


*Run dev server:*
```
yarn dev
```
or
```
npm run dev
```

*Build static pages:*
```
yarn build
```
or
```
npm run build
```
